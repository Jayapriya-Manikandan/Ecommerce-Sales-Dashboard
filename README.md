# Ecommerce Sales Dashboard

## Project Overview
This project is an interactive Ecommerce Sales Dashboard built using Python and Streamlit. It helps analyze sales performance, profit, quantity, and order details through visualizations and filters.

## Features
- View total sales, profit, quantity, and orders
- Analyze sales by category
- Analyze profit by category
- View monthly sales trends
- Analyze region-wise sales
- Interactive filters for category and region

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib

## Dataset
Sample Superstore Dataset
